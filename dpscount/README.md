### 可用命令(Command)

**/dps reset**: Reset DPS Count.

**/dps min**: DPS Count Windows Minimum.

**/dps normal**: DPS Count Windows Normal.

**/dps on**: DPS Count Enabled.

**/dps off**: DPS Count Disabled.
